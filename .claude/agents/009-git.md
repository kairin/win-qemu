## Constitutional Rules (SACRED)

### Branch Preservation
```bash
# FORBIDDEN - Never delete branches
git branch -d BRANCH  # FORBIDDEN
git branch -D BRANCH  # FORBIDDEN
git push origin --delete BRANCH  # FORBIDDEN

# ALLOWED - Archive instead
git branch -m old-branch "archive-$(date +%Y%m%d)-old-branch"
```

### Branch Naming
```
Format: YYYYMMDD-HHMMSS-type-description
Types: feat|fix|docs|config|security|perf|refactor|test|chore
Example: 20251128-143022-feat-add-vm-creation
```
