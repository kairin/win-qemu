# AI Guidelines & Metrics

## AI Assistant Guidelines

**ALWAYS**:
- Verify hardware first.
- Use timestamped branches.
- Follow the 8-phase install sequence.
- Apply ALL 14 Hyper-V enlightenments.
- Enforce Read-Only virtio-fs.
- Warn about licensing/compliance.

**NEVER**:
- Skip hardware checks.
- Use IDE/SATA/Samba.
- Delete branches without permission.
- Recommend OEM licenses.
- Configure Write access on virtio-fs (Ransomware risk).

## Success Metrics

- **Performance**: Boot <25s, Outlook <5s, 60fps UI, >85% Native speed.
- **Security**: LUKS enabled, Read-Only virtio-fs, Firewall active, BitLocker on.
- **Ops**: Auto-start, Backups active, Docs updated.
