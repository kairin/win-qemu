Installation Guide

Complete setup guide for QEMU/KVM Windows 11 virtualization on Ubuntu 25.10

Prerequisites

CPU with VT-x/AMD-V + IOMMU • 16GB+ RAM • Ubuntu 25.10 • NVMe SSD

Step 1: System Verification

First, verify your system supports hardware virtualization:

system-check.sh 
"text-base-content/50"># Check CPU virtualization support
grep -cE '(vmx|svm)' /proc/cpuinfo
"text-base-content/50"># Output should be > 0 (number of CPU cores)

"text-base-content/50"># Check IOMMU groups (required for GPU passthrough)
ls /sys/kernel/iommu_groups/
"text-base-content/50"># Should show numbered directories (0, 1, 2, etc.)

"text-base-content/50"># Check available RAM
free -h
"text-base-content/50"># Minimum 16GB recommended

Step 2: Install QEMU/KVM Packages

Install all required virtualization packages:

install.sh 
"text-base-content/50"># Clone the repository
git clone https://github.com/kairin/win-qemu.git
cd win-qemu

"text-base-content/50"># Run master installation script (requires sudo)
sudo ./scripts/install-master.sh

"text-base-content/50"># This installs:
"text-base-content/50"># - qemu-kvm (virtualization engine)

"text-base-content/50"># - ovmf (UEFI firmware)
"text-base-content/50"># - swtpm (TPM 2.0 emulator)
"text-base-content/50"># - And 5 more packages...

Important: Reboot after installation for group membership changes to take effect.

Step 6: Post-Installation Driver Setup

After Windows 11 installation completes, install VirtIO drivers for optimal performance.

Important: Display Resolution Fix

Windows will initially show only 1280x800 resolution because it uses "Microsoft Basic Display Adapter". Install the VirtIO GPU driver to enable resolutions up to 4K.

VirtIO GPU Driver Installation

  1. Open Device Manager (Win+X, select Device Manager)
  2. Expand "Display adapters"
  3. Right-click "Microsoft Basic Display Adapter"
  4. Select "Update driver" â†’ "Browse my computer for drivers"
  5. Navigate to: D:\viogpudo\w11\amd64\
  6. Click "Next" and allow driver installation
  7. Reboot Windows
